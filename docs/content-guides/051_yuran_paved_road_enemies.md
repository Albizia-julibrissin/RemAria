# 遊覧舗装路跡の敵定義（エネミーマスター用）

spec/050_enemy_master.md に基づく、遊覧舗装路跡エリアの敵マスタ定義。  
Seed や実装時にこのドキュメントを参照する。

---

## 共通ルール

- **配置**: 前列 = (row: 1, col: 1)、中列 = (row: 2, col: 2)、後列 = (row: 3, col: 3)。
- **ステータス**: 7基礎ステ（STR, INT, VIT, WIS, DEX, AGI, LUK）の合計 = CAP。各ステは CAP の約 10～30% で配分（「高め」= 25～30%、「低め」= 10%、「超低め」= 10%）。
- **スキル**: 既存 Skill を名前で参照。Seed で Skill は name 一意で投入されている前提。
- **アイコン**: 仮で全敵 `iconFilename: "0.gif"` を参照。

## 備考

- **浄化の祈り（敵側）**: 対象「味方のうちHP割合最低の1人」は、敵が複数体いるときはその中の1体を回復。中ボス単体のときは自分自身。
