## コンテンツ追加ガイド概要

コンテンツ追加ガイドは、**「スキル追加」「探索エリア追加」など、ゲーム内コンテンツを増やすときの実務フロー**をまとめたドキュメント群です。

- **docs/**: 仕様やアイデア、設計メモ。わたしとあなたで読み合わせ・合意する場所。
- **spec/**: 実装のための設計書。実装担当（AI）が主に読む。正本の API / データ仕様。
- **docs/content-guides/**: 上記 2 つを前提にした「実務マニュアル」。コンテンツを追加・調整するときの手順書。

基本ルールは次の通りです。

- コンテンツを増やすときも、**まず docs / spec 側で仕様・設計を整理してから実装に入る**。
- 具体的な「手順」「チェックリスト」は、この `content-guides` 配下のガイドに追記していく。
- スキルや探索エリアなど、**種別ごとに 1 ファイル**を用意し、あとから追記・更新していく。
