# 機能一覧と優先度（MVP中心）

## 優先度の定義
- MVP：最初に動かす最小セット
- Next：MVP後に追加（運用/拡張に必要）
- Later：後回し（ギルド/市場など）

---

## MVP（まず動かす）

### アカウント
- [ ] ユーザー登録/ログイン
- [ ] セッション管理
- [ ] プロフィール（表示名・称号程度）

### キャラ管理
- [ ] アカウントキャラ作成
- [ ] 仲間1体の所持/選択（厳選は後で拡張でも可）
- [ ] メカ1体の所持（初期は基本形）
- [ ] ステータス表示（STR/INT/DEX/VIT など暫定）
- [ ] 装備枠（武器/防具…）と基礎値反映
- [ ] 遺物枠（4枠）と効果表示（生成・厳選の詳細はNextでも可）

### 訓練（時間経過育成）
- [ ] 訓練設備の建設（最低2種＋調整1種）
- [ ] キャラを設備に配置
- [ ] 経過時間でステ変動（上げ下げ）
- [ ] CAP（総合上限）の概念
- [ ] 完了/進行の表示、履歴ログ

### 探索（ソロPvE）
- [ ] 探索出撃（パーティ3体固定）
- [ ] 戦闘（自動ログ）→勝敗→報酬
- [ ] ドロップ（素材/遺物/設計図など暫定）
- [ ] 消耗（耐久/弾薬など：まずは耐久だけでも可）
- [ ] 帰還

### 工業（生産）
- [ ] 資源の所持/消費
- [ ] 装備の製造（武器/防具の基礎値）
- [ ] メカパーツの製造（最初は少数）
- [ ] 設備の建設/更新（訓練設備含む）
- [ ] 倉庫/在庫

---

## Next（MVP後）

### 戦闘の拡張
- [ ] 作戦UI（優先度・条件・ターゲットの拡張）
- [ ] 状態異常（スチパン/魔法寄り）
- [ ] 乱数シード固定（検証・リプレイ用）

### 育成の拡張
- [ ] 仲間の性格/適性
- [ ] ステ解釈分岐（力→攻撃/運搬など）

### 工業の拡張
- [ ] 設備の種類追加（偵察/キャンプなど）
- [ ] 設計図/研究ツリー

### PvP（ソロ）
- [ ] 決闘
- [ ] トーナメント

---

## Later（後回し）

### 市場（経済）
- [ ] 出品/購入
- [ ] 手数料
- [ ] 価格履歴

### ギルド
- [ ] ギルド作成/加入
- [ ] ギルド戦（シーズン・週末）
- [ ] ガーディアン（本拠点メカ）

---

## 仕様/Specへのリンク方針
- 各MVP機能は `spec/` に1枚ずつ作る（例：`spec/010_training.md`）。
- `docs` は“地図”、`spec` は“契約（入出力とルール）”。
